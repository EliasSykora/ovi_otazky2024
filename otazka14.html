Přímý potomek selektor1 > selektor2 { } - označí jen potomka na nejvyšší úrovni Pseudotřídy Elementy ovládané myší nebo klávesnicí Poznáme je podle : :hover - najetí myši :active - nakliknutý objekt :visited - navštívený odkaz Pseudoelementy Označují části objektů Poznáme je podle :: selektor::typ p::first-letter - první písmeno odstavce Vkládání drobného obsahu: h1::before { content: text; } h1::after { content: text; } Selektory podle vlastností selektor[vlastnost=hodnota] a[target="_blank"] - vybere všechny odkazy do nového okna Vlastnosti CSS: https://www.w3schools.com/cssref/index.php vlastnost: hodnota; Pro objekt s 4 hranami: margin: 10px; - všechny 4 hrany současně margin: 10px 5px; - nastaví současně horní + dolní, pravou + levou margin: 10px 5px 20px 5px; - každou zvlášť margin-top: 20px; - jen jednu Vlastnost s procentuálními hodnotami filter: opacity(1); 1 = 100% 0.5 = 50% Jednotky px - obrazové body/ pixely % - procenta velikosti nadřazeného objektu vw, vh - procenta velikosti okna em - základní jednotka fontu, přizpůsobuje se přiblížení v systému Co umí CSS nastavit width height opacity font-weight font-family font-decoration color - barva textu background background-color background-image background-position display: block, flex, grid, inline, none position: static, relative, absolute, fixed, sticky trasform transition padding CSS animace Tvorba animace: @keyframes nazev { from { vlastnost:hodnota; } to { vlastnost:hodnota; } } @keyframes nazev { 0% { vlastnost:hodnota; } 50% { vlastnost:hodnota; } 100% { vlastnost:hodnota; } } Detaily se nastaví v objektu: animation: nazevAnimace trvani opakovani; animation: jump 1s loop;
